Using Masks for Editing and Modeling

In IFTDSS shapefiles can be used to select specific areas for editing, or running models. This can be particularly helpful in working with treatment areas, units, etc. To use this feature you must first have a shapefile available for your project area. For information on creating or uploading shapefiles see the About Shapefiles Topic.

Masks can be used in IFTDSS to define an area of interest when generating a landscape summary, or applied during the Edit Landscape task to apply rules only to a specific part of your landscape.

In the Landscape Comparison task, you may use masks during the editing process to represent fuel treatments, then compare the results. An example of this is can be worked through by following the Comparing Treatment Alternatives Tutorial.

Masks are not currently supported for individual landscape fire behavior runs.
