Calculating Landscape Fire Behavior for Landscape Evaluation

Landscape Fire Behavior takes you to the Modeling Playground, where you may select and run fire behavior for your landscape.

Generating the Fire Behavior Model

 

Generating a fire behavior model will create an output layer in Map Studio that will display:

  • Flame Length (meters)
  • Spread Rate (m/min)
  • Intensity kW/m)
  • Heat Area (kJ/m2)
  • Crown State (crown fire activity, No Fire, Surface Fire, Passive Fire)

Modeling outputs will be stored in Modeling Playground, where they may be added to Map Studio, or used to generate a summary report containing tables, charts, and visual representations of your model outputs.

To generate the fire behavior model:

Step 1: Give your run a descriptive name.

Step 2: Select a landscape for your run from the dropdown menu. You also have the option of

creating a new landscape from within the run interface

To create a landscape within the Generate Landscape Summary task, select the Create Here option.

  1. Use the zoom tools in the top left of the map to zoom to your desired location
  2. Click the Select Draw Tool, mouse to your location on the map, hold down the left mouse key, and draw a rectangle around your desired area, releasing the button when you are finished.
  3. Alternatively, if you wish to enter exact coordinates rather than draw a box around your area, you may do so by entering the coordinates, in degrees, for the west, east, south, and north bounds of your desired area.
  4. Select the version of LANDFIRE data you would like to use.
  5. Select the fire behavior fuel model (Fuel Model 40 or Fuel Model 13) you would like included in your landscape file.
  6. Give your landscape a descriptive name, this will be helpful as you accumulate more landscape files in IFTDSS.
  7. Select the folder in which you would like to store your landscape file, or select the create new folder button to create and name a new folder in which to place your landscape file.
  8. Select Create to generate your landscape file

Once you select Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, will contain the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass will be replaced by a green checkmark.

The maximum area for a landscape is 12,188,160 acres (138 miles x 138 miles).

or creating a new landscape from within Map Studio

To create landscape from within Map Studio you will use the options in the Landscape Tools button, under the Create new tab.

  1. Use the Map Studio tools such as zoom, basemap layers, etc. to locate your desired area.
  2. Click the Select draw mode, mouse to your location on the map, hold down the left mouse key, and draw a rectangle around your desired area, releasing the button when you are finished.
  3. Alternatively, if you wish to enter exact coordinates rather than draw a box around your area, you may do so by entering the coordinates, in degrees, for the west, east, south, and north bounds of your desired area.
  4. Select the version of LANDFIRE data you would like to use.
  5. Select the fire behavior fuel model (Fuel Model 40 or Fuel Model 13) you would like included in your landscape file.
  6. Give your landscape a descriptive name, this will be helpful as you accumulate more landscape files in IFTDSS.
  7. Select the folder in which you would like to store your landscape file.
  8. Select Create to generate your landscape file.

After selecting Create, it may take a few minutes to generate your landscape. You may check its status by refreshing the Layer List in Map Studio.

If you have created a landscape in Map Studio using the Create in Map Studio button (for example, this is an option in Landscape Summary, and other tasks) you will need to navigate back to your task to continue that process using your newly created landscape.

The maximum area for a landscape is 12,188,160 acres (138 miles x 138 miles).

When your landscape is selected, you will notice a box appears displaying details about the selected landscape, including, source data, boundaries, and acres.

 

Next, select the heading for each model input type, and populate the fields in the dropdown:

Step 3: Specify windspeed, in miles per hour, at 20 feet above the ground or above the vegetation (allowed values 0-100 mph).

Enter wind direction as the direction the wind is blowing from (in degrees clockwise from north, with north being 0 or 360 degrees). Alternatively you may enter -1 for upslope wind, or -2 for downslope wind.

Step 4: Specify a crown fire calculation method, and the foliar moisture content (in percent).

Step 5: Specify fuel moisture, in percent, for:

  • 1 hour fuels (1hr FM) - allowed values 2-100
  • 10 hour fuels (10hr FM) - allowed values 2-100
  • 100 hour fuels (100hr FM) - allowed values 2-100
  • Herbaceous fuels (Herb) - allowed values 30-250
  • Live Woody fuels (WoodyFM) - allowed values 60-200

If you would like to specify fuel moisture by fuel model, you may do so by adding rows, select the + add row option. This will add a new line. Using the drop down list, select a fuel model, then specify the fuel moisture values. You may repeat this as many times as needed.

If you choose to only populate one row, those fuel moistures will be applied to all fuel models in your landscape.

Step 6: Upon completing all required inputs, you may choose to:

Save - which will save your inputs, but still provide a chance to edit and re-save before running the model.

Save and Run - which will save your inputs and run the model.
