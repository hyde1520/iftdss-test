Swipe

During various analyses you may want to visually compare two landscapes, or compare model run outputs such as fire behavior with various landscape attributes.. This is where the Swipe tool comes in handy. The Swipe tool allows you to click and drag a bar across the Map Studio screen to view layer interactions.

To use the tool, select Swipe from the top right of Map Studio.

Choose the layers you wish to swipe between. You may swipe between masks/polygons, landscapes, or model runs outputs.

In the tool dropdown the layer you select will be visible on the top part of the slider, the other layer will be visible on the bottom part of the sliders.

Layers and features must already be turned on Map Studio either via the Layer List or Simulation Output Tools buttons before they are visible in the Swipe dropdown menu.
