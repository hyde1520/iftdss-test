Saving Shapefiles for Future Use

Currently, this extra step is required to save an uploaded shapefile in My Workspace for later use in IFTDSS. Without this step, navigating away from Map Studio results in an uploaded shapefile not being saved. This is a known issue that will be corrected in a later IFTDSS release.

To retain your uploaded shapefiles in IFTDSS if you follow the steps below:

Upload a zipped shapefile, ensure you select which attributes you would like to include (you may choose up to 10).

When completed, a confirmation message will appear to verify the shapefile has been uploaded

To select the new shapefile, exit Add Shapefile and open Layer List to zoom to your new shapefile, by

  1. selecting the icon to the right of your shapefile and
  2. selecting the Zoom to option

Open Edit Tools and select the Copy from Existing Features button

Select the polygon you wish to save (selected polygons will turn yellow). This can be done by either selecting individual polygons or selecting and dragging a box over the polygons you wish to retain.

if you plan on selecting multiple polygons, select the Select mulitpart option under the copy from Existing Features button .

Left click any remaining polygons you wish to save as part of your retained shapefile.

Currently you must select Copy from Existing Features button before highlighting each additional polygon.

Once all polygons you wish to retain have been selected, enter a name for your shapefile and choose a folder location for the polygons to be saved. Saved polygons will then be available in My Workspace and viewable in Map Studio for future use throughout IFTDSS.
